# Contribute to Mrapi

## Development Workflow

1. Fork & clone
1. Install the dependencies

   ```bash
   yarn
   ```

1. Start a watch server

   ```bash
   yarn watch
   ```

## Link packages & use

- link
   ```bash
   cd packages/create-mrapi-app
   yarn link
   ```
- use
  ```bash
  cd path/to/empty/folder
  yarn create mrapi api-demo
  ```
