## 环境依赖

**Node:**

- NodeJS >= 10.x
- NPM >= 6.x

**Database:**

- MySQL >= 5.6
- PostgreSQL >= 10
- SQLite >= 3

## 快速开始

新建一个 mrapi 项目

```bash
yarn create mrapi-app my-project

# or
npx create-mrapi-app my-project

# or
pnpx create-mrapi-app my-project
```

以 DEV 模式启动应用程序

```bash
npm start
```
